📤 Google Sheets Integration

Set up a Google Apps Script webhook to sync time records automatically. Every clock in/out and AUX change will be posted to your sheet.

function doPost(e) { var d = JSON.parse(e.postData.contents); var ss = SpreadsheetApp.openById('YOUR_SHEET_ID'); var sheet = ss.getSheetByName('Logs') || ss.insertSheet('Logs'); if (sheet.getLastRow() === 0) sheet.appendRow(['Date','Emp ID','Name','AUX','Timestamp']); sheet.appendRow([d.date, d.empId, d.name, d.aux, d.timestamp]); return ContentService.createTextOutput('OK'); }
